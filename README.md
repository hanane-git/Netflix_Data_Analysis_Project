# 🎬 Netflix Data Analysis Project

## 📌 Overview
This project focuses on performing Exploratory Data Analysis (EDA) on the Netflix dataset to extract meaningful insights about movies and TV shows available on the platform.

The goal is to analyze trends, content distribution, and patterns using data visualization and Python libraries.

---

## 📊 Problem Type
This is a **Data Analysis (EDA)** project.

No machine learning models are used in this project. The focus is on understanding data through statistics and visualization.

---

## 📁 Dataset
The dataset used in this project is the Netflix Movies and TV Shows dataset.

- Source: Kaggle
- File: `netflix_titles.csv`
- Contains information about movies and TV shows including:
  - Title
  - Type (Movie / TV Show)
  - Country
  - Release Year
  - Genre
  - Rating

---

## ⚙️ Workflow

The project follows a standard data analysis pipeline:

1. Load dataset
2. Explore data structure
3. Handle missing values
4. Data cleaning
5. Exploratory Data Analysis (EDA)
6. Data visualization
7. Extract insights

---

## 📊 Key Analysis Performed

- Distribution of Movies vs TV Shows
- Top content-producing countries
- Most common genres
- Content trend over years
- Missing data analysis

---

## 📈 Visualizations

The following visualizations were created:

- Bar charts for content type distribution
- Top countries producing Netflix content
- Most popular genres
- Trend of content release over time

---

## 🧠 Key Insights

- Movies are more frequent than TV Shows on Netflix
- The United States produces the highest amount of content
- Drama and Comedy are the most common genres
- Netflix content production has increased significantly after 2015
